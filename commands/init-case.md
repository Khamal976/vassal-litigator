# init-case -- Инициализация дела

Когда пользователь запускает эту команду с "$ARGUMENTS":

1. Прочитай скилл `skills/init-case/SKILL.md` и следуй его инструкциям.
2. Если `$ARGUMENTS` содержит описание дела (стороны, суть, номер, суд, имя клиента и т.п.) -- извлеки из него поля и используй как стартовые данные для фазы 1 (не переспрашивай то, что уже дано).
3. Если аргументов нет -- собери информацию о деле в свободной форме (см. SKILL.md → фаза 1).
4. Выполни полный pipeline из SKILL.md: pre-flight -> сбор данных -> создание структуры папок -> case.yaml -> пустой index.yaml -> history.md -> сводка Сюзерену -> опциональный sync-notion.
